Email Chat Bot

This is a personal project used for managing my sports camp email address. It monitors the inbox and replies to FAQ by querying the OpenAI API and a context CSV file. It runs on google cloud console and uses a pub/sub notification ping system to run the code. It has a python backend that you can read in the repository below. Email Brooksportscamp@gmail.com a common question; i.e. what are the hours?
