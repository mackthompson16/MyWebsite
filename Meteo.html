Mars Meteo

This was my team's project for our freshman Hackathon. We created a 'Mars Weather App' that pulls data off of the Curiosity and Perseverance Rovers and displays it on our website. The likes of our coding skills involved solving leetcode problems and making simple games, but we had never implemented a real website before. We learned how to operate GitHub and tie Frontend and Backend elements together. We used simple JS, HTML, and CSS styling.
